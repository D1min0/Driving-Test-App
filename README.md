# 🚗 Test Drive Booking Website

A modern and responsive web application that allows users to browse available vehicles and schedule a test drive through a simple and intuitive interface. The project focuses on delivering a clean user experience while demonstrating responsive design and front-end development best practices.

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- php
- xampp

---

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and clean user interface
- Interactive vehicle showcase
- Test drive booking form
- Smooth navigation between sections
- Form validation
- Fast loading and optimized layout

---

## 👤 What Users Can Do

- Browse available vehicles
- View vehicle information
- Select a preferred test drive date
- Fill out and submit a booking request
- Navigate the website easily across different devices

---

## ⚙️ Development Process

The project started by planning the website structure and user flow. Wireframes were created before implementing the layout with HTML and CSS. JavaScript was then used to add interactivity and improve the overall user experience. Finally, the project was tested across different screen sizes to ensure responsiveness and consistency.

---

## 🏗 How It Was Built

The website was developed using semantic HTML for structure, CSS for styling and responsive layouts, and JavaScript for interactive functionality such as form handling and user interactions. Version control was managed with php, and the project is hosted through Xampp and Netlify.
